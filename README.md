# 자기소개
#### 광주_1반_강대수

## 1. ![이미지](https://edu.ssafy.com/asset/images/header-logo.jpg) **SSAFY에 입과하게 된 과정** 
  -  전자공학을 전공하여 전기기사를 준비 중 이었으나, 적성과 맞지 않는다고 생각하였고 SW 개발에 관심이 생겨 관련 프로그램을 찾던 중 SSAFY에 대하여 알게 되었고 지원하게 되었습니다.
  
---

## **2. 코딩 경력**
- 대학교 재학 중 간단한 예제 수행 경험
- SSAFY를 통해 Python, Django 학습
- Spring Boot를 활용하여 홀로 어르신 케어 서비스, 전자 동화책 프로젝트 진행

## **3. SSAFY에 입과한 후 포부**
 - 비전공자로써 입과 했지만 SSAFY 과정을 포기하지 않고 수료하여 한사람의 개발자로 성장하고 싶습니다.
